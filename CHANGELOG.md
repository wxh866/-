# 更新日志 Changelog

本文档记录截图工具的所有重要版本变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

---

## [4.0] - 2026-06-20

### 新增
- 全新 UI 设计，深色/浅色双主题一键切换
- 统一主题色彩管理系统（`theme_colors.py`）
- 工具栏悬浮/固定双模式
- 截图历史记录面板
- 4 种截图模式：全屏、区域、窗口、滚动
- 10 种标注工具：画笔、直线、矩形、圆形、箭头、文字、马赛克、高亮、橡皮擦、水印
- 一键保存/分享操作按钮
- WCAG AA 色彩对比度标准
- 设计规范文档（`design-spec.md`）
- UI 交互原型（`screenshot-ui-prototype.html`）

### 优化
- 主题切换即时生效，所有组件同步刷新
- 精简 UI 布局，移除冗余元素
