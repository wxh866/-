# 使用AI写的截图软件
## 截图工具 功能总结
### 一、截图模式
- 全屏截图 — 一键截取整个屏幕（多显示器支持），支持 Ctrl+Shift+S 全局快捷键
- 区域截图 — 拖拽选择任意区域，支持多显示器 DPI 缩放适配
  - 选区可拖拽移动、边缘调整大小
  - 双击空白区 = 全屏截图，单击空白区 = 清除选区
  - 边缘自动吸附，实时显示选区像素尺寸
  - Enter 确认 / Esc 取消

### 二、编辑器标注工具（8种）
|工具|功能|
|----|----|
|✏️ 画笔|自由手绘，支持鼠标滚轮调节粗细|
|📏 直线|拖拽绘制直线|
|▭ 矩形|拖拽绘制矩形边框|
|◯ 圆形|拖拽绘制椭圆|
|➤ 箭头|拖拽绘制带箭头线条|
|T 文字|点击放置文字，自定义字号|
|◌ 橡皮擦|擦除标注内容，恢复原图|
|㊙ 水印|平铺/居中文字水印，可调透明度、旋转、颜色、字号|

### 三、通用功能
- 撤销 / 重做 — 最多 30 步历史（Ctrl+Z / Ctrl+Y）
- 颜色选择 — 10 种预设颜色 + 自定义取色器
- 线宽调节 — 1-30px，+/- 按钮或鼠标滚轮
- 保存 — PNG/JPEG/BMP 格式，自定义保存路径
- 复制到剪贴板 — Ctrl+C 一键复制
- 缩放适配 — 图片自动适配窗口大小，Ctrl+0 恢复初始比例
- 深色主题 — 统一的暗色 UI 设计

### 四、其他
- 浮动工具栏 — 无边框、可拖拽、始终置顶
- 设置面板 — 保存路径、热键、默认文件格式
- 帮助对话框 — 完整的使用说明和快捷键列表
- 单文件打包 — 约 17.8MB 的独立 .exe，无需安装 Python
