# 截图工具 v4.0 · UI 设计规范

> 像素君 出品 · 2026.06.20
> 双主题现代化截图软件界面设计系统，覆盖设计令牌、组件库、响应式策略、无障碍标准与交互规范。

---

## 一、设计原则

| 原则 | 说明 |
|------|------|
| **简洁现代** | 扁平化视觉，去除冗余装饰，让截图内容本身成为焦点 |
| **语义化图标** | 工具图标无需文字即可理解，降低认知负荷 |
| **零等待响应** | 截图触发后 100ms 内呈现选区，工具切换 150ms 过渡 |
| **双主题自适应** | 深色/浅色主题共享同一套设计令牌，颜色对比度均达 WCAG AA |
| **像素级精准** | 选区支持方向键 1px 微调，尺寸实时显示 |

## 三、核心组件库

### 3.1 截图模式标签（Mode Tabs）

四档模式：**区域截图 / 全屏截图 / 窗口截图 / 滚动截图**

- 容器：圆角 10px 卡片，4px 内边距
- 选中态：强调色填充，白色文字
- 未选中：透明背景，悬停浮起
- 图标 + 文字组合，图标 16px

### 3.2 选区框（Selection Region）

截图区域的核心视觉元素：

- 描边：1.5px 实线，强调色
- 遮罩：选区外区域半透明黑罩（深色 45% / 浅色 32%）
- 8 个手柄：四角 + 四边中点，11×11px 白底蓝边方块
- 尺寸标签：选区顶部，强调色背景，等宽字体
- 支持拖拽移动、八向缩放、方向键 1px 微调

### 3.3 悬浮标注工具栏（Annotation Toolbar）

主工具行包含 10 个工具，分四组：

| 组别 | 工具 | 快捷键 |
|------|------|--------|
| 标注 | 箭头、矩形、文字、马赛克、高亮、椭圆、画笔 | A / R / T / M / H / O / P |
| 自定义 | 颜色与粗细 | C |
| 编辑 | 撤销、重做、复制 | Ctrl+Z / Ctrl+Y / Ctrl+C |
| 确认 | 确认保存（强调色） | Enter |

**双模式支持：**
- **悬浮模式（默认）**：工具栏漂浮于选区下方，跟随选区移动，自动避让屏幕边界
- **固定模式**：吸附屏幕右侧常驻，纵向排列，不遮挡操作区。按 `F` 切换

**工具单元格规范：**
- 尺寸 34×34px，圆角 7px
- 图标 18×18px，1.6–2px 描边
- 选中态：强调色填充
- 悬停：浮层背景 + tooltip 提示（含快捷键）

### 3.4 颜色与粗细面板（Custom Panel）

点击颜色工具展开：

- **颜色行**：12 色圆形色板（22px 直径）+ 自定义取色器（彩虹环 + 加号）
- **粗细行**：4 档预设（2/4/6/10px），圆点大小对应实际粗细
- 选中态：色板白边 + 焦点环；粗细项淡蓝背景

### 3.5 预览与编辑面板

- 缩略预览：16:9 占位区，棋盘格底
- 元信息网格：尺寸 / 大小 / 格式 / 缩放，2×2 卡片布局
- 操作按钮：保存（强调色主按钮）+ 分享（幽灵按钮）

### 3.6 历史记录面板

- 2 列网格，16:10 缩略图
- 悬停浮起 + 强调描边
- 左下角时间徽章（半透明黑底）
- 点击加载历史截图

### 3.7 状态栏

底部固定，实时显示当前工具 / 颜色 / 粗细，右侧列出全局快捷键提示。

## 四、交互规范

### 4.1 截图触发流程

```
按下 Ctrl+Shift+S
  → 100ms 内屏幕变暗，进入选区模式
  → 拖拽确定区域（或选择模式自动捕获）
  → 释放鼠标，选区框 + 工具栏 + 尺寸标签呈现（150ms 淡入）
  → 标注编辑阶段
  → Enter 确认保存 / Esc 取消
```

### 4.2 工具切换

- 鼠标点击：即时切换，工具栏不收起
- 快捷键：单字母（A/R/T/M/H/O/P）直接切换，无需修饰键
- 切换动画：150ms 颜色过渡

### 4.3 选区操作

| 操作 | 方式 |
|------|------|
| 移动选区 | 选区内拖拽 |
| 调整大小 | 拖拽 8 个手柄 |
| 像素微调 | 方向键 ←↑↓→（1px/次） |
| 快速微调 | Shift + 方向键（10px/次） |
| 重选 | 工具栏"重选"按钮 |
| 居中 | 工具栏"居中"按钮 |

### 4.4 快捷键体系

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+S` | 触发截图 |
| `A` / `R` / `T` / `M` / `H` / `O` / `P` | 切换标注工具 |
| `C` | 展开/收起颜色粗细面板 |
| `F` | 切换工具栏悬浮/固定模式 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Y` | 重做 |
| `Ctrl+C` | 复制截图到剪贴板 |
| `Enter` | 确认保存 |
| `Esc` | 取消截图 |

## 五、响应式策略

### 5.1 断点

| 断点 | 宽度 | 布局调整 |
|------|------|----------|
| 大屏 | ≥1280px | 三栏完整布局，工具栏横向 |
| 中屏 | 980–1279px | 右侧面板压缩至 280px |
| 平板 | 640–979px | 单栏布局，右侧面板横向排列 |
| 手机 | <640px | 模式标签换行滚动，工具栏自动折行 |

### 5.2 分辨率适配

- 所有尺寸采用相对单位 + 像素混合，保证高 DPI 屏幕锐利
- 选区手柄最小 11px，满足触控 44px 目标的等效命中区
- 工具图标 SVG 矢量，任意缩放不失真
- 棋盘格背景使用 CSS 渐变，无图片依赖

## 六、无障碍标准（WCAG AA）

| 项目 | 标准 | 实现 |
|------|------|------|
| 文字对比度 | ≥ 4.5:1 | 深色 #F0F0F8/#0F0F1A = 16.8:1；浅色 #1A2233/#FFF = 15.3:1 |
| 大文字对比度 | ≥ 3:1 | 强调色对白底 4.6:1 达标 |
| 焦点可见 | 清晰焦点环 | `--ring` 3px 半透明环 |
| 键盘可达 | 全功能键盘操作 | 所有工具/模式/确认均有快捷键 |
| 触控目标 | ≥ 44px | 工具格 34px + 命中区扩展 |
| 动效敏感 | 尊重系统偏好 | 支持 `prefers-reduced-motion` |
| 色彩非唯一信息 | 图标 + 文字 tooltip 双重编码 | 工具均含语义图标与文字提示 |

## 七、视觉规范要点

1. **选区边框清晰可见**：1.5px 强调色实线 + 外部遮罩，确保任意背景下可辨识
2. **工具图标语义化**：箭头用斜线+箭头头、马赛克用网格、高亮用马克笔形，无文字依赖
3. **状态反馈即时**：所有交互 150ms 内响应，悬停态明确
4. **深浅主题一致**：两套主题共享结构与间距，仅颜色令牌切换
5. **信息密度适中**：工具栏单行容纳全部工具，避免二级菜单嵌套
